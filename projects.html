DOLPHIN CUSTOMER RELATONS MANAGEMENT SYSTEM

DOLPHIN CRM OVERVIEW

Dolphin CRM is a complete, role-based Customer Relationship Management (CRM) system built from the ground up using PHP, MySQL, JavaScript, HTML, and CSS. The project demonstrates secure backend architecture, relational database design, and dynamic frontend interactions using AJAX.The system is designed to help teams manage customer relationships efficiently by organizing contacts, tracking communication through notes, and enforcing role-based access control for security and accountability.

WORLD DATABASE LOOKUP ENGINE

LOOKUP ENGINE OVERVIEW

This project involves the development of a dynamic web service designed to query and retrieve geographical data from a relational database. The application serves as a lightweight search engine, allowing users to look up statistics about countries and their associated cities without reloading the page.

AVENGERS CHARACTER SEARCH ENGINE

SUPERHERO SEARCH OVERVIEW

This project involves building a dynamic search tool that interfaces with a PHP backend to display superhero biographies. The project focuses on using AJAX to create a seamless user experience by updating the interface without page reloads.

VOLUNTEER MANAGEMENT SYSTEM

VOLUNTEER SYSTEM OVERVIEW

Designed and implemented a modular Java application to manage non-profit volunteer operations. The system is built on a strict Layered Architecture (MVC-inspired), separating data access, business logic, and user interaction to ensure code maintainability and scalability.
